# Proyecto Ingeniería de Software 2
## Idea inicial:

Aplicación para gestionar comisiones a artistas freelance. El artista puede mostrar sus obras al público y ser contactado por usuarios convencionales para hacer pedidos personalizables.
Cada artista tendrá en su perfil sus obras con diferentes valoraciones y comentarios de otros usuarios (por ver).

Grupo 5: Alberto Morán Reina, Maxim Berchun, Alonso Carrera Martínez, Jorge Alonso Fernández.
